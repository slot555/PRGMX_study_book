Font-Awesome

Font Awesome — это шрифт с иконками, которые могут добавляться к любым элементам веб-страницы, чтобы повысить их наглядность и улучшить дизайн.

Установка

Перейдите на сайт Font Awesome и скачайте архив с последней версией (в более ранних версиях установка была значительно проще, кому интересно Гугл в помощь).

Прямая ссылка на скачивание (если версия обновилась ссылка может быть сломана).

Распаковываем архив в папку с проектом в котором хотим их использовать. Далее необходимо подключить 3 javascript файла в head раздел html файла, в котором мы им будем использовать.

                    <script defer src="path_to_fontawesome_folder/js/brands.js"></script>
                    <script defer src="path_to_fontawesome_folder/js/solid.js"></script>
                    <script defer src="path_to_fontawesome_folder/js/fontawesome.js"></script>
У меня получилось: 
                    <script defer src="../../../../fontawesome-free-5.15.4-web/js/brands.js"></script>
                    <script defer src="../../../../fontawesome-free-5.15.4-web/js/solid.js"></script>
                    <script defer src="../../../../fontawesome-free-5.15.4-web/js/fontawesome.js"></script>

Для использования иконок достаточно указать их тег. Например инструкцию для подключения иконки звезды можно посмотреть на сайте. На странице вы увидите саму иконку и инструкцию к их подключению, которая выглядит так: 

                    <i class="fas fa-star"></i>
это и есть тот тег который будет отображать иконку звезды.

Transition

Transition позволяет определять переходное состояние между двумя состояниями элемента. Различные состояния могут быть определены с помощью псевдоклассов, таких как :hover или :active. Длугими словами это плавный переход между крайними состояниями растянутый по времени, которое может указать программист.

Пример

html 

                    <div class="transition_example">
                        <p>some text</p>    
                    </div>

css 

                    .transition_example{
                        width: 300px;
                        height: 100px;
                        background-color: blueviolet;
                        transition: 0.5s;
                    }
                    .transition_example:hover{
                        width: 500px;
                        transition: 0.5s;
                    }

Transform

CSS3-трансформации позволяют сдвигать, поворачивать и масштабировать элементы. Трансформации преобразовывают элемент, не затрагивая остальные элементы веб-страницы, т.е. другие элементы не сдвигаются относительно него.

Тут подробная техническая информация.

Пример

html 

                    <div class="transform_example">
                        <p>some text</p>    
                    </div>

css 

                    .transform_example{
                        width: 200px;
                        height: 200px;
                        background-color: chartreuse;
                    }
                    .transform_example:hover{
                        transform: skewX(30deg);
                    }

Images CSS Background

Вставка картинок как задний фон через CSS 

                    background: url("../img/universe.jpg") transparent no-repeat center center  / cover;
помогает более просто настроит их адаптивность, но это плохая практика, т.к. если Ваш шаблон будет использоваться программистом для динамичного изменения контента то работать с CSS намного сложнее чем с HTML.
Более подробную информацию по background можно почитать ТУТ.

Пример

html 
                <div class="background_img_example"></div>
css 
                .background_img_example{
                    width: 200px;
                    height: 200px;
                    background: url("../img/200x200.gif") transparent no-repeat center center  / cover;
                }

Images (HTML Picture)

Использование тега picture позволяет почти полностью отказаться от использования css, и подменять изображение в зависимости от разрешения монитора, чтобы уменьшил объем загружаемого через сеть контента.

Пример

html 
                <div class="picture_tag_example">
                    <picture>
                        <source srcset="img/marvel.jpg" type="image/jpeg" media="(min-width:1330px)">
                        <source srcset="img/dc.jpg" type="image/jpeg" media="(min-width:992px)">
                        <source srcset="img/starwars.png" type="image/png" media="(max-width:992px)">
                        <img src="img/marvel.jpg" alt="">
                    </picture>
                </div>
css 
                .picture_tag_example{
                    width: 100%;
                    text-align: center;
                }
                .picture_tag_example img{
                    max-width: 100%;
                }

Images (HTML object-fit)

Свойство object-fit определяет, как содержимое заменяемого элемента, такого как <img> или <video>, должно заполнять контейнер относительно его высоты и ширины.

Для object-fit можно указать одно из нижеперечисленных свойств: fill, contain, cover, none, scale-down.
Подробнее ТУТ

Пример

html 
                <div class="object_fit_example">
                    <div class="object_fit_example_row">
                        <div class="object_fit_example_col">
                            <div class="object_fit_example_item">
                                <img src="img/marvel.jpg" alt="">
                            </div>
                        </div >
                        <div class="object_fit_example_col">
                            <div class="object_fit_example_item">
                                <img src="img/dc.jpg" alt="">
                            </div>
                        </div>
                        <div class="object_fit_example_col">
                            <div class="object_fit_example_item">
                                <img src="img/starwars.png" alt="">
                            </div>
                        </div>
                    </div>
                </div>
css 
                .object_fit_example{
                    margin: 0;
                }
                .object_fit_example_row{
                    display: flex;
                }
                .object_fit_example_col{
                    flex: 0 1 33.333%;
                    padding: 0px 15px;
                }
                .object_fit_example_item{
                    margin: 0px 0px 20px 0px;
                    padding: 0px 0px 58% 0px;
                    overflow: hidden;
                    position: relative;
                }
                .object_fit_example_item img{
                    width: 100%;
                    height: 100%;
                    object-fit: cover;
                    position: absolute;
                    top: 0;
                    left: 0;
                }
