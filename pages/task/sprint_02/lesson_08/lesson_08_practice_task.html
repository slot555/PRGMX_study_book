Lesson_08. Images. Font-Awesome..

Задание

Подготовительный этап

  1. Переделать indexстраницу проекта (страницу меню) на блочную модель с использованием FLEX и добавить новый урок.

Задание урока

    Используя севжеизученный материал сверстать блоки со следующим содержимым:
  1. Используя Font Awesome сверстать маркированный список с произвольными маркерами иконками. Также к блоку применить CSS свойство transition изменять блок при наведении (псевдокласс :hover).
  2. Используя CSS свойство Transform сверстать блок с подвижными элементами.
  3. Сверстать блок и вставить в него изображение задним фоном.
  4. Сверстать блок и вставить в него изображение используя тег picture.
  5. Сверстать блок, состоящий из трех блоков, размещенных горизонтально и вставить в них картинки и текст.
  6. Сверстать блок, состоящий из трех блоков, размещенных вертикально и вставить в них картинки и текст.
  7. Вся верстка с картинками должна быть адаптивной.
