Lesson 17. Frameworks - Bootstrap. Intro.

Задание

Подготовительный этап

  1. Добавить новый урок в меню.

Задание урока:

  1. Используя Bootstrap 5 сверстать товарные карточки для какого-либо товара (напрмер ноутбуки).
  2. Карточка товара предполагает наличие:
    1. картинки товара;
    2. описание товара;
    3. цена товара;
    4. кнопка фирмы;
    5. кнопка просмотра подробностей;
    6. кнопка добавления в корзину;
    7. если есть желание, можно добавить дополниельные элементы, но они должны быть уместны на карточке товара.
  3. Сделать не менее 4-х товаров.
  4. Дополнить/изменить хотя-бы один базовый Bootstrap класс своими настройками.
