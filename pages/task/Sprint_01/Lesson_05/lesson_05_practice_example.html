Git commands

git clone https://github.com/s..... - Клонировать репозиторий на локальный компьютер
git pull - Получить все изменения из репозитория
git checkout -b branchname - Создать новую ветку и переключится на нее
git add . - Добавить изменения, которые сделали локально
git commit -m "commit text" - Подписать изменения
git push origin branch_name - Закачать последний коммит на сервер
